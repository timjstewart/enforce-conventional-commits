# Prerequisites

- bash
- git

# Installation

1. In a terminal, navigate to this local repository's directory.
1. Run:

```
installGlobally.sh
```

Every time you run `git init` the `commit-msg` hook script will be added to the new repository's git hooks (i.e. `.git/hooks`)
